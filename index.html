Cost Effective Field Research Device Using Open Sourced Hardware

Materials List Wiring Code for Ardunio Giga Code for Google Sheets

The limited technical knowledge in research fields like biology, ecology and agriculture is a barrier in making custom affordable equipments for data collection. However, current boom in openhardware microcontrollers like Arduio, RasberryPi has open many opportunites in making and deploying devices that are cheap and easy to make. Here I am going to show you how to use an Arduino Giga to collect data from a sensor that uses using SDI-12 (Serial Digital Interface at 1200 baud) protocol and upload it to googlesheets.
